# Orbit AI — WEBSITE & PLATFORM CATALOG
*Exhaustive menu of products, layouts, features, options, and development services — tier-gated. The ladder mirrors the KB: **Gold → Platinum → Iridium → Rhodium.** Premium add-ons open at higher tiers; the premium-of-premium options live only at the top.*

> **Two axes.** **Build tier** (Gold→Rhodium, §A–F) = *what you get*. **Service level** (DIY · Hybrid/Guru-Assisted · DFY · Diamond Support, §G) = *how it's delivered & supported*. A client picks one of each. Premium-of-premium is gated on both: **Rhodium** (build) and **Diamond** (support).

## G. SERVICE & SUPPORT LEVELS (how it's delivered)

The build tier (§A–F) defines **what you get.** The service level defines **how much we do and how much support you get.** They're independent axes — a client picks **one build tier × one service level.**

| Level | Tagline | Who does the work | Best for |
|---|---|---|---|
| **DIY** | Self-Serve Kit | You build with our tools + docs | agencies, resellers, technical owners |
| **Hybrid** | **Guru Assisted** | You drive; a Guru co-pilots the hard parts | owners who want control + expert hands |
| **DFY** | Done For You | We build, deploy & hand over | busy owners who want it handled |
| **Diamond** | **Diamond Support Package** | Dedicated Full Architect Guru + priority queue | market leaders who want it owned + evolving |

### What each level includes

| Pillar | DIY | Hybrid (Guru Assisted) | DFY | Diamond (add-on) |
|---|---|---|---|---|
| Site build | templates + generators, you deploy | you build, Guru does complex parts + reviews | we build & deploy end-to-end | + proactive redesign / optimization |
| KB / agent | packs + compiler, you wire | guided compile + wiring | compiled + wired by us | + ongoing prompt tuning |
| Dialect | glossary + packs, you select | Guru tunes register / dialect | selected + tuned by us | + dialect re-validation cycles |
| Automation | workflow templates, you import | Guru wires the tricky flows | built + tested by us | + new automations on request |
| Onboarding | docs / knowledge base | scheduled working sessions | full handover + training | + quarterly strategy reviews |
| Support / SLA | standard queue (async) | standard + Guru sessions | standard post-launch | **Priority Queue + Full Architect Guru** |

### The Guru ladder (how much expert you get)
- **DIY** → Guru via docs / community only.
- **Hybrid** → **Guru Assisted:** scheduled sessions; the Guru unblocks you and builds the complex parts (dialect, compliance, automation).
- **DFY** → the Guru's team builds it *for* you.
- **Diamond** → a **dedicated Full Architect Guru:** proactive monitoring + optimization, direct line, first access to new features.

### Diamond Support Package (premium of premium)
The apex support layer — **gated to higher build tiers** (add-on on **Iridium**, **included by default on Rhodium**):
- **Priority Support Queue** — front of the line, defined fast-response SLA.
- **Full Architect Guru** — a dedicated architect assigned to the account: proactive system health + optimization, monthly evolution/strategy, direct line (chat/phone), first access to new dialects/features, quarterly audits + dialect re-validation.

### The two-axis model
```
                  DELIVERY / SUPPORT  →
                  DIY     Hybrid    DFY     Diamond
   B  Gold        ✓       ✓         ✓       —
   U  Platinum    ✓       ✓         ✓       —
   I  Iridium     ✓       ✓         ✓       add-on
   L  Rhodium     ✓       ✓         ✓       included
   D  ↓
```
**Build tier = what · Service level = how.** Standard support on Gold/Platinum, Priority on Iridium, **Diamond at the top.** Diamond is premium-of-premium and cannot be bought on Gold/Platinum — same ladder logic as the build features.

## H. How to price it (framework — set your own numbers)
Two pricing axes stack:
- **Build fee** scales with the **build tier** (Gold < Platinum < Iridium < Rhodium) — sets the *scope*.
- **Delivery** scales with the **service level** (DIY lowest → Hybrid → DFY highest) — sets the *labor*.
- **Recurring:** hosting + maintenance + receptionist minute pool + automation runtime — **plus a Diamond Support uplift** where selected.
- **À-la-carte add-ons:** per the tier-gated table in §D; premium-of-premium (build) carries a **Rhodium** floor, premium-of-premium (support) = **Diamond**.
- **Anchor on ROI, fixed price, no surprises** (clients are burned by agencies that scope-creep $10k→$25k). Quote the build flat; meter only the receptionist minutes/usage.

> Worked example: *Iridium build × DFY delivery + Diamond Support* = full premium site/receptionist, built end-to-end by Orbit, with a dedicated Architect Guru and priority SLA — your top everyday package below Rhodium.

*Set final dollar figures in your offer deck — this catalog defines the menu, the build tiers, the service levels, and the gating on both axes.*
