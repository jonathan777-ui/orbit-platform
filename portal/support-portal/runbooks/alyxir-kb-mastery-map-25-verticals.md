# Alyxir KB Mastery Map — Top 25 Verticals × Sub-Niche

The spine of the world-class library. Every vertical is a **base layer** (shared persona, compliance, intent map, intake, schema, bilingual glossary) with **sub-niche overlays** that tune voice, add niche FAQs/intake fields, and set urgency rules. Overlays may **tighten** compliance, never loosen it. Each sub-niche line: *signature intent · key intake · urgency/sensitive*.

Legend: **✦ deep base built** (live in the Merge Engine) · ○ queued. Compliance = the hard line the agent never crosses.

## Tier A — highest call volume / heaviest bilingual demand

### 1. Home Services ✦ (HVAC built; tree queued)
Register: warm, plain, Spanglish common. **Compliance:** safety hazards (gas/CO/flood/no-heat-in-freeze) → safety script + 911/utility + emergency dispatch; no binding quote/diagnosis by phone.
- **HVAC** ✦ — no heat/AC, maintenance, install · address, system age, symptom · no heat in freeze / no AC + vulnerable occupant → priority
- **Plumbing** ○ — leak, clog, water heater, no water · address, fixture · active flooding/burst/sewage → emergency + shut-off guidance
- **Electrical** ○ — outage, panel, install · address, symptom · sparking/burning/exposed wire → safety + emergency
- **Roofing** ✦ — leak, storm, replacement · property address, roof type/age · active interior water → URGENT
- **Landscaping / lawn** ○ — quote, recurring · lot size, frequency · none
- **Cleaning** ○ — one-time/recurring, move-out · beds/baths/sqft, date · none
- **Pest control** ○ — infestation, recurring · pest, area · stinging insects near allergic person → priority
- **Garage door** ○ — won't open, spring, install · symptom · car/person trapped → priority
- **Appliance repair** ○ — brand, symptom · brand/model · none
- **Pool** ○ — service, repair, open/close · pool type · none

### 2. Automotive ✦
Register: plain-warm, honest, "no upsell games." **Compliance:** no phone diagnosis as fact, no guaranteed quotes/HP, no street-legal/emissions promise, hard fitment gate; safety (brakes/steering/smoke/accident) → don't drive, tow, 911 if injury.
- **Auto repair / mechanic** ○ — symptom, quote, appt · year/make/model, mileage · brake/steering failure → tow
- **Performance / tuning** ✦ — parts, build, dyno · YMMT, current mods, goal · fitment gate before any commitment
- **Body / collision** ○ — estimate, insurance, drivable? · damage, claim · not drivable/airbags → tow + insurance path
- **Tire / wheel** ○ — flat, set, alignment · vehicle, size · blowout/stranded → roadside/tow
- **Detailing** ○ — package, appt · vehicle, package · none
- **Dealership — service** ○ — maintenance, recall, appt · VIN/recall · none

### 3. Restaurants & Hospitality ✦ (full-service built)
Register: warm-casual, on-brand. **Compliance:** never promise a dish is allergen-free (capture + flag kitchen); alcohol/age per policy; no guaranteed wait time.
- **Full-service** ✦ — reservation, large party · date/time, party, allergies, occasion · allergy → kitchen flag
- **Fast-casual / QSR** ○ — order, hours, delivery · items, pickup time · none
- **Pizza / delivery** ○ — order, ETA, deal · address, items · none
- **Bar / nightlife** ○ — table/bottle service, events, cover · date, party, age · ID policy
- **Catering** ○ — off-site quote · date, headcount, cuisine, dietary · event date firm
- **Food truck** ○ — location today, catering · event/date · none

### 4. Real Estate ○
Register: warm-professional, market-savvy. **Compliance:** no legal/financing/appraisal advice or value guarantees; **fair-housing — never steer by protected class.**
- **Residential buyer** — see listing, pre-approval, tour · budget band, area, timeline, pre-approved?, has agent? · ready-today buyer → priority
- **Residential seller / listing** — home value, list · address, timeline, condition · CMA not a guaranteed price
- **Luxury** — concierge, discreet · tactful qualification · none
- **Commercial** — lease/buy/invest · use, size, budget, area · none
- **New construction** — community, move-in · builder, timeline · none
- **Rentals / leasing** — availability, apply · dates, budget · fair-housing strict

### 5. Property Management ○
Register: plain, courteous. **Compliance:** no landlord-tenant legal advice; fair-housing; habitability emergencies escalate.
- **Residential PM** — maintenance request, rent, owner inquiry · property/unit, issue · triage owner/tenant/prospect
- **Maintenance / work orders** — repair request · unit, issue · no heat/water, flood, gas, lockout → emergency dispatch
- **Leasing / tenant** — availability, apply, showings · dates · none
- **HOA / community** — dues, violations, amenities · account · route board matters
- **Commercial PM** — tenant services, building issues · suite, issue · safety → now

## Tier B — professional services (high-gravity, `usted`-default)

### 6. Law Firms ○ (gold-standard base exists as reference)
Register: premier, discreet, trauma-aware. **Compliance:** no legal advice, no fee/outcome promises, conflict screening, privilege; emergencies (arrest, ICE detention, active DV, imminent deadline) → URGENT live handoff.
- Personal Injury · **Immigration** ✦ · Family/DV · Criminal Defense · Workers' Comp · Estate & Probate · Business · Real Estate law · Bankruptcy · Employment

### 7. Immigration Law ✦ (broken out — heaviest bilingual niche)
**Compliance:** no legal advice / "do you have a case", no fee/outcome promise, status never a barrier; ICE detention = emergency (name, A-number, facility).
- **Family-based** ✦ — petition, green card · relationship, status · deadline
- **Citizenship / naturalization** ○ — eligibility, application · time as resident · test/interview date
- **Removal / deportation defense** ○ — hearing, detention · hearing date, A-number · detention → EMERGENCY
- **Asylum** ○ — application, deadline · entry date · 1-year deadline → priority
- **DACA** ○ — renewal, first-time · current status · renewal window
- **Work / employment visas** ○ — H-1B, employer petition · employer, role · filing window

### 8. Accounting, Tax & Bookkeeping ○
Register: precise, professional-warm. **Compliance:** no tax advice/positions, no refund/approval guarantees; **never collect SSN/bank/card over open voice — secure link.**
- **Individual tax prep** — file, refund, ITIN · tax year, W-2/1099 count, ITIN need · ITIN/immigration confidentiality
- **Tax resolution / IRS problems** — letter, audit, lien, back taxes · notice type+date, amount · IRS deadline → priority; high anxiety
- **Small-business bookkeeping** — monthly, catch-up · entity, software, months behind · none
- **Payroll** — run payroll, new employee · # employees, states · filing deadlines
- **CFO / advisory** — strategy, forecasting · revenue band · none

### 9. Insurance Agencies ○
Register: protective, plain-spoken. **Compliance:** no coverage/eligibility advice or rate guarantees; active claim w/ injury → escalate, don't assign fault; CMS-marketing rules (health/Medicare).
- **Auto** — quote, SR-22, claim · vehicles, drivers, ZIP · SR-22/DUI no judgment
- **Home / property** — quote, bundle, claim · property, loss type · active loss → claim + restoration referral
- **Life** — term/whole quote · beneficiary · gentle re health/mortality
- **Health / Medicare** — enrollment, ACA/Medicare · SEP · enrollment-window deadline; no plan-steering
- **Commercial / business** — GL, WC, BOP, COI · industry, payroll · none

### 10. Financial & Wealth Advisory ○
Register: dignified, fiduciary-feel. **Compliance:** no investment/tax advice, no return promises.
- **Retirement planning** — 401k rollover, readiness · age band, accounts · none
- **Investment advisory** — portfolio review, new account · investable band, goals · none
- **Estate & wealth transfer** — trust, legacy · coordinate estate attorney · dignified
- **Debt / credit counseling** — debt relief, budgeting · situation · shame/anxiety → reassure

### 11. Mortgage & Lending ○
Register: clear, "no surprises." **Compliance:** no rate/approval guarantees; secure app link for sensitive data.
- **Purchase / home loan** — pre-approval, rates, first home · timeline, area, credit band, down payment · none
- **Refinance** — lower rate, cash-out · current loan, goal · rate-lock/closing deadline → priority
- **FHA/VA/first-time** — low-down, VA eligibility · program info, not advice · none
- **Reverse mortgage** — senior-focused · patient, dignified, no pressure · none
- **Commercial / hard money** — investor/bridge · property, use, exit · none

## Tier C — health & wellness (HIPAA-minded, gentle)

### 12. Medical & Dental ✦ (dental built)
Register: caring, gentle, `usted`. **Compliance:** no diagnosis/treatment/med advice; HIPAA-minded; emergent symptoms → 911/ER, never triage.
- **Dental — general** ✦ — cleaning, pain, broken tooth · insurance, DOB · trauma/swelling/bleeding → same-day/ER
- **Dental — ortho** ○ — braces/Invisalign, payment plans · age, goal · none
- **Oral surgery** ○ — extraction, implant · referral · pre-op instructions
- **Primary care** ○ — new patient, refill, results · reason, insurance, DOB · chest pain/breathing/stroke → 911
- **Pediatrics** ○ — well-child, sick visit · parent, symptom · fever/breathing → ER
- **OB/GYN** ○ — prenatal, annual · privacy · bleeding/pain → nurse line/ER
- **PT / chiro** ○ — eval, plan of care · injury, referral · none

### 13. Aesthetics & Med Spa ✦
Register: luxe, discreet, confidence-building. **Compliance:** no efficacy/candidacy claims; defer to provider consult; separate medical vs non-medical.
- **Injectables (Botox/filler)** ✦ — pricing, consult, touch-up · goal area, first-time?, event date · no medical promise
- **Laser / skin** ○ — hair removal, resurfacing, packages · skin concern, prior treatments · none
- **Body contouring** ○ — CoolSculpting consult · goal area · never body-shaming
- **Medical dermatology** ○ — acne, lesions, skin check · concern · changing/bleeding lesion → prompt provider
- **Lash / brow / non-medical** ○ — booking, fills · service · casual-warm

### 14. Behavioral Health & Therapy ○
Register: safe, nonjudgmental, slow. **Compliance:** no diagnosis/triage; **suicidal/risk → crisis path + warm human handoff (988), never just a booking;** HIPAA-minded.
- **Individual therapy** — new client, insurance/sliding scale, specialty match · concern (brief), insurance, modality · risk statement → crisis line
- **Psychiatry / med eval** — meds management · referral · no med advice
- **Couples / family** — multi-participant scheduling · participants · none
- **Substance use / rehab** — intake, detox, levels of care · situation · overdose/withdrawal danger → 911; confidentiality
- **Child / adolescent** — parent intake · mandatory-reporter awareness (route, don't probe)

### 15. Veterinary ○
Register: compassionate, pet-as-family. **Compliance:** no diagnosis/med advice; poisoning/trauma/bloat → emergency vet + poison control.
- **General vet** — wellness, vaccines, sick · species/breed, age, symptom, pet name · none
- **Emergency / ER vet** — urgent · symptom · hit-by-car/poison/can't-breathe/bloat → immediate; bring toxin packaging
- **Specialty (surgery, oncology)** — referral-based · gentle
- **Mobile vet** — service-area scheduling
- **Grooming / boarding (add-on)** — vaccine records required

## Tier E — service businesses (expansion set to reach 25)

### 21. Funeral, Cremation & Memorial ○
Register: deeply compassionate, unhurried, dignified. **Compliance:** grief-aware (never rush/upsell), no pre-need price guarantees beyond authorized GPL, FTC Funeral Rule awareness; at-need death call → immediate compassionate handoff.
- **At-need (immediate)** — death occurred, transport · location of deceased, name, callback · immediate human handoff
- **Pre-need planning** — pre-arrange, pricing · plan interest · gentle, no pressure
- **Cremation** — direct cremation, urns · service preference · none
- **Memorial / celebration of life** — venue, scheduling · date, headcount · none

### 22. Childcare & Early Learning ○
Register: warm, reassuring to parents. **Compliance:** never share another family's info; ratio/licensing facts only; mandatory-reporter awareness (route, don't probe); collect minors' info per policy.
- **Daycare / infant care** — enrollment, availability, ratios · child age, schedule needed, start date · waitlist status
- **Preschool / pre-K** — enrollment, curriculum, tour · age, program · enrollment window
- **After-school / camps** — registration, hours · grade, dates · none
- **Drop-in / hourly** — availability · age, date · none

### 23. Education, Tutoring & Test Prep ○
Register: encouraging, results-minded but honest. **Compliance:** no score/admission guarantees; minors policy (parent intake); accommodations captured, not advised.
- **K-12 tutoring** — subject help, assessment · grade, subjects, goals · falling-behind → priority scheduling
- **Test prep (SAT/ACT/GRE)** — packages, diagnostic · test date, target · test date → urgency
- **Language learning** — placement, classes · level, goal · none
- **Music / arts lessons** — instrument, scheduling · level, instrument · none
- **Special education / accommodations** — support match · needs (route to specialist) · gentle

### 24. Moving & Storage ○
Register: dependable, transparent, calming (moving is stressful). **Compliance:** binding vs non-binding estimate clarity, no firm price without survey, damage-claim path; long-distance regulatory awareness.
- **Local moving** — quote, date, crew size · origin/destination, bed count, date · move date firm
- **Long-distance / interstate** — quote, timeline · cities, inventory size, date · DOT/regulatory aware
- **Packing services** — add-on quote · scope · none
- **Storage units** — availability, size, price · unit size, move-in date · climate-control need
- **Commercial / office moves** — quote, after-hours · size, timeline · none

### 25. Pet Services (non-medical) ○
Register: warm, pet-as-family, playful-professional. **Compliance:** vaccine records required; no veterinary diagnosis/medical advice (route to vet); injury/illness on premises → owner + vet.
- **Grooming** — booking, breed, services · breed, coat, last groom · matting/skin issue → note
- **Boarding / daycare** — availability, vaccine reqs · dates, vaccine records, temperament · vaccine proof required
- **Dog training** — class, behavior consult · age, issue, goals · aggression → in-person eval
- **Pet sitting / walking** — recurring, meet-and-greet · address, schedule, pet info · key/access handling
- **Mobile grooming** — service-area, scheduling · location · none
